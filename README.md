# Proeza

**Uma solução de Banco de Dados Relacionais**. Repositório destinado ao desenvolvimento de uma plataforma web para análise de dados públicos da Câmara dos Deputados.

## Sobre o Projeto

O projeto consiste em uma aplicação web voltada para visualização e análise de dados provenientes da API de Dados Abertos da Câmara dos Deputados. A plataforma permite consultar informações sobre deputados, partidos, estados e despesas parlamentares, apresentando indicadores e visualizações gráficas que auxiliam na interpretação dos dados públicos. A solução é composta por um backend desenvolvido em Python responsável pela coleta, processamento e disponibilização dos dados através de uma API REST, e um frontend moderno desenvolvido em Next.js para exibição das análises.

### Fonte de Dados

* API de Dados Abertos da Câmara dos Deputados
